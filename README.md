# Building AI Systems

Solution for lab-01 that can be used as the starting point for subsequent labs, particularly the reuse of `config/llm_config.py`.

To get started:

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.sample .env
python config/llm_config.py 
```

Tested with python 3.14
